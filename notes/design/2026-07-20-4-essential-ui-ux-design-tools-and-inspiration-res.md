# ארבעה כלי עזר ואתרי השראה למעצבי UI/UX

## תקציר

אוסף של ארבעה כלים ואתרי השראה ממוקדים למעצבי UI/UX המקצרים את תהליך העבודה. האוסף כולל כלי ליצירת מוקאפים אסתטיים למסכים, גלריית השראה ייעודית לעיצובי פוטרים, מחולל גרדיאנטים תלת-ממדיים מונפשים הנתמך ב-Figma ו-Framer, ופלטפורמת השראה המתמקדת כולה בממשקים במצב כהה (Dark Mode).

## תובנות מפתח

- שימוש בגלריות נישה ממוקדות (כמו השראה לפוטרים בלבד או עיצובי Dark Mode בלבד) מאפשר למצוא פתרונות עיצוביים מדויקים במהירות רבה יותר מאשר בגלריות עיצוב כלליות.
- שילוב גרדיאנטים דינמיים ומונפשים משדרג את הוויזואליות של אתרים מודרניים, וכלים חדישים מאפשרים לייצא אותם ישירות כקוד React או כאלמנטים בתוך כלי עיצוב.

## פריטי פעולה

- [ ] לשמור בסימניות את Footer.design ו-Dark.design לשימוש בשלב מחקר המשתמש וההשראה (Moodboarding).
- [ ] לנסות את Shots.so ליצירת מוקאפים מהירים ומעוצבים עבור מצגות או תיק עבודות.
- [ ] לבדוק את התוסף של ShaderGradient ב-Figma או Framer להוספת רקעים דינמיים לעיצובים הבאים.
